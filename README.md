# SvelteKit Starter

Starter template with:

- SvelteKit
- TypeScript
- Tailwind CSS
- shadcn-svelte
- Supabase
- Zod
- svelte-i18n
- Vitest

## Planned structure

```txt
src/
├── lib/
│   ├── auth/
│   ├── components/
│   ├── i18n/
│   ├── server/
│   ├── stores/
│   ├── utils/
│   └── validators/
├── features/
├── routes/
└── tests/
```
